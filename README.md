# Jerry's GitHub Homepage

这是 Jerry 的个人 GitHub Pages 主页源码。

## 如何编辑

- 修改 `index.html`：改名字、介绍、项目、联系方式。
- 修改 `style.css`：改颜色、字体、间距和页面样式。
- 修改 `script.js`：改网页交互逻辑。

## 如何上传照片

1. 准备一张照片，建议裁剪成正方形。
2. 把照片命名为 `avatar.jpg`。
3. 放进 `assets` 文件夹。
4. 上传到 GitHub 时，确保仓库里也有 `assets/avatar.jpg`。

如果你想用 PNG 图片，可以把文件命名为 `avatar.png`，并把 `index.html` 里的 `assets/avatar.jpg` 改成 `assets/avatar.png`。

## 如何发布

1. 在 GitHub 创建仓库：`jerry4919.github.io`
2. 把本文件夹里的所有文件上传到仓库根目录。
3. 打开仓库的 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，Folder 选择 `/root`。
6. 等待 1 到 3 分钟，访问 `https://jerry4919.github.io/`。
